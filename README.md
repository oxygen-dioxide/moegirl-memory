本仓库收集我认为萌娘百科上值得收藏的历史页面，例如[铁道梗](铁道梗/20220908)。

如果你有想收藏的历史页面，可以提出issue，我会**酌情**考虑收藏。但请注意：
- 只有当前仍能访问的历史版本页面才能收藏。
- 涉及政治、年龄限制内容，或侵犯版权的页面不予收藏。
- 即使以上条件均通过，我也会视情况决定是否收藏。
